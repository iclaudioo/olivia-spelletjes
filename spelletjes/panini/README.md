# Olivia's Panini WK 2026 Companion 🏆

Digitale sticker-tool voor Olivia op `/spelletjes/panini/`.

Wat erin zit:
- Dashboard met totale voortgang en collector level
- Alle 48 landen met stickerstatus
- Teamdetail met 20 aanklikbare stickers per land
- Group Battle en werelddeel-radar
- Superstar Tracker
- Ruilcentrum voor dubbele stickers
- Backup export/import via JSON
- Foto-preview voor nieuwe stickerfoto's

Alles wordt lokaal op het toestel bewaard met `localStorage`.
